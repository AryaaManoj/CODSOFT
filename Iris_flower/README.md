This project aims to classify the iris flower into 3 species - setosa, versicolor and virginica using a kaggle dataset consisting of 150 records of their measurements.
The dataset is from "sklearn.datasets".
Files included:
 'iris_dataset.csv' – cleaned dataset
 'iris_model.pkl' – trained Logistic Regression model
 'iris_summary.md' – result summary
 'notebook.ipynb' – full code with training, evaluation, and plots
